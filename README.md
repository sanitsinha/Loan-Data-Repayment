# Loan Repayment Prediction

## Overview
This project aims to predict whether a borrower will repay a loan or default using machine learning techniques. The focus is on building a reliable classification model by analyzing financial and demographic features.

The project covers the complete workflow from data preprocessing to model evaluation, providing practical exposure to real-world predictive modeling problems.

---

## Key Features
- Performed data cleaning and preprocessing, including handling missing values and encoding categorical variables  
- Conducted Exploratory Data Analysis (EDA) to identify patterns and relationships in the data  
- Trained multiple machine learning models for classification  
- Evaluated model performance using metrics such as accuracy, precision, recall, and F1-score  
- Compared different models to select the most effective approach  

---

## Dataset
The dataset consists of borrower-related information such as income, loan amount, credit history, and other financial attributes.

Preprocessing steps included:
- Handling missing and inconsistent data  
- Encoding categorical features  
- Scaling numerical variables where required
